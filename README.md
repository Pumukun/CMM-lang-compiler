# LGPP. C-- language and compiler
## Build
### Linux
```
$ git clone https://github.com/Pumukun/LGPP.git
$ make
$ ./bin/g-- "file_path"
```

### Windows 
```
$ git clone https://github.com/Pumukun/LGPP.git
$ g++ -Wall -I include -c src/*.cpp -o obj/*.o
$ g++ obj/*.o -o bin/g--
$ cd bin
$ g-- "file_path"
```
